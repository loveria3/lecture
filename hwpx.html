스킬 설치

다운받은 HWPX 스킬 저장하는 방법

1
스킬 파일 다운로드
위의 ⚡ HWPX 스킬 다운로드 버튼을 눌러 hwpxskill-main.zip 파일을 저장합니다. 압축 해제하지 않고 ZIP 그대로 사용합니다.
2
Claude 데스크탑 앱 → 사용자 지정 → 스킬
Claude 데스크탑 앱 좌측 사이드바에서 사용자 지정을 클릭하고, 스킬 탭을 선택합니다.
3
+ 버튼 → 스킬 만들기 → 스킬 업로드
스킬 목록 우측 상단의 + 버튼 클릭 → 스킬 만들기 → 스킬 업로드를 선택하고, 다운받은 hwpxskill-main.zip 파일을 그대로 업로드합니다. (압축 해제 불필요)
4
설치 완료 확인
Skills 목록에 HWPX 스킬이 표시되면 설치 완료! 이제 Cowork 탭에서 한글 문서 작업을 요청할 수 있습니다.
💡
스킬은 claude.ai 웹과 Claude 데스크탑 앱(Cowork 탭) 모두에서 동작합니다. Free · Pro · Max · Team · Enterprise 플랜 모두 사용 가능해요. 단, 설정 → 기능 → 코드 실행 및 파일 생성이 켜져 있어야 합니다.
선생님 실습

샘플 파일 업로드 후 프롬프트 예시

샘플 파일을 다운받아 Claude Cowork 탭에 업로드한 뒤, 아래 프롬프트를 그대로 입력해 보세요.

⚠️
HWPX 파일은 Claude 화면 안에서 미리보기가 되지 않습니다.
Claude가 파일을 생성하면 다운로드 링크가 제공됩니다. 파일을 내려받은 뒤 한컴오피스(한글) 또는 한컴 뷰어에서 열어 결과를 확인하세요.
샘플 1

업무 보고서 작성

프롬프트 업로드한 업무 보고서 양식을 기반으로 완성된 한글 파일을 만들어줘. - 보고 제목: 2학기 학생 생활지도 현황 보고 - 보고일: 2026년 9월 5일 - 담당: 생활지도부 주요 현황: - 지각 23건, 결석 7건 (9월 1~5일 기준) - 생활규정 위반: 복장 불량 12건, 휴대폰 사용 8건 - 학교폭력 신고: 0건 보고 배경, 목적, 향후 계획은 학교 현장에 맞게 내용을 보완해서 공문서 형식의 문어체로 작성해줘.

샘플 2

행사 계획서 작성

프롬프트 업로드한 행사 계획서 양식을 기반으로 완성된 한글 파일을 만들어줘. - 행사명: 2026 제주고 가을 축제 「탐라의 빛」 - 일시: 2026년 10월 16일(금) 10:00~17:00 - 장소: 본관 중앙홀 및 운동장 - 대상: 전교생 및 학부모 - 주관: 학생자치회 세부 일정: 10:00 개회식 / 10:30 부스 운영 / 13:00 무대 공연 / 15:00 시상 및 폐회 행사 목적, 기대 효과, 역할 분담은 학교 행사 특성에 맞게 보완해서 세부 일정은 표로, 전체는 공문서 격식체로 작성해줘.
