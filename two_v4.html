강의

3. 강의 — Skills vs MS Office 애드인, 언제 뭘 쓰나?

오늘 배울 도구는 두 가지입니다. 둘 다 Claude를 쓰지만 사용 상황이 다릅니다. 이 차이를 먼저 알면 이후 실습이 훨씬 명확해집니다.

🖥️ SKILLS (Cowork 탭)
"새 파일을 만든다"
  • Claude가 처음부터 파일을 생성
  • 결과물이 .docx · .xlsx · .pptx · .pdf로 저장됨
  • 내 폴더에 바로 파일이 떨어짐
💬 "체육대회 가정통신문 Word 파일로 만들어줘"
→ .docx 파일이 새로 생성됩니다 ✅
✔ 언제 쓰나: 아직 파일이 없을 때 · 새로 만들어야 할 때
📎 MS OFFICE 애드인
"열린 파일을 편집한다"
  • Excel·PPT 파일을 열어 놓은 상태에서 Claude 사용
  • 현재 파일 내용을 Claude가 자동으로 읽음
  • 분석·수정·개선 제안을 즉시 적용
💬 성적표.xlsx를 열고 "이 데이터 평균 이하 학생 표시해줘"
→ 현재 시트에 바로 반영됩니다 ✅
✔ 언제 쓰나: 이미 파일이 있을 때 · 기존 자료를 수정·분석할 때
📊 한눈에 비교
구분 🖥️ Skills (Cowork) 📎 MS Office 애드인
실행 위치 Claude 데스크탑 앱 Cowork 탭 Excel · PPT 앱 안 사이드패널
주요 기능 새 파일 처음부터 생성 열린 파일 분석·수정·개선
파일 형식 .docx · .xlsx · .pptx · .pdf · .hwpx Excel(.xlsx) · PowerPoint(.pptx)
설치 방법 설정 → 기능 → 코드 실행 ON Excel/PPT → 추가기능 → Claude 검색
⚙️ Skills 활성화 (한 번만) — 설정(⚙️) → 기능 → 코드 실행 및 파일 생성 ON 이 설정 하나로 Word · Excel · PPT · PDF 파일을 모두 만들 수 있습니다. 이미 설정하신 분은 그대로 진행하세요.
⚙️ 실제 설정 화면
설정(⚙️) → 기능 → 코드 실행 및 파일 생성
→ 토글 ON
이 설정이 꺼져 있으면 Skills가 파일을 생성하지 못합니다.

강사 시연

7. 강사 시연 — MS Office 애드인 통합 활용

Excel과 PowerPoint 안에서 Claude를 직접 설치하면 현재 파일의 내용을 자동으로 인식해 분석·수정·생성이 가능합니다.

✅ Claude 사이트에서 별도 설정 없음 — 커넥터·MCP·플러그인 연결 불필요. Claude 계정 로그인만 하면 됩니다.
설치는 Excel 안에서 바로 할 수 있습니다. "엑셀 안에서 설치하는 앱"이라고 생각하면 됩니다.
설치 방법 (Excel · PPT 동일)
1
Excel 또는 PPT를 열고 [홈] 탭 클릭
2
[추가기능] 버튼 클릭
3
검색창에 "Claude" 입력 후 설치
4
Claude 계정으로 로그인 → 우측 패널에 Claude 창 열림
한 번 설치로 Excel/PPT 모두 사용 Excel에 설치하면 PPT에서도 동일하게 사용할 수 있습니다.
통합 컨텍스트 활용
현재 열린 파일의 내용을 Claude가 자동으로 읽습니다.
파일 내용을 별도 복사 없이 바로 분석·요약할 수 있습니다.
수정 제안을 받아 직접 파일에 적용할 수 있습니다.
활용 예시 Excel 성적표를 열고 "이 데이터에서 평균 이하 학생을 표시해 줘."라고 입력하면 즉시 처리됩니다.
교원 업무 활용 분야
Excel: 성적 분석, 출결 현황 정리, 통계 자동 생성
PPT: 수업 자료 초안 생성, 슬라이드 수정·보완
절약 효과 반복적인 데이터 입력·정리 작업을 자연어 지시 하나로 대체할 수 있습니다.

선생님 실습

8. 선생님 실습 — PPT 플러그인으로 교육 자료 작성

PPT 파일을 열어 두고 Claude에 지시하면 슬라이드 내용을 자동으로 생성하거나 개선 제안을 받을 수 있습니다.

✏️ 복사는 하되 [ ] 대괄호(슬라이드 번호·단원명·학년)는 직접 바꿔 입력하세요.
04
PPT 플러그인 20분

수업 자료 슬라이드 작성·수정·시각화

실습 A: 슬라이드 내용 개선

현재 PPT 파일의 [3번] 슬라이드를 분석해서:

1. 핵심 내용을 3줄로 요약해 줘.
2. 학생이 더 이해하기 쉬운 표현으로 바꿔 줘.
3. 제목을 더 임팩트 있게 수정해 줘.
4. 내용을 시각적으로 표현할 도식 구성을 제안해 줘.

실습 B: 새 슬라이드 생성

다음 내용으로 수업용 PPT 슬라이드를 만들어 줘.

- 주제: [단원명] 핵심 개념 정리
- 학년: [학년]
- 포함 내용:
  · 핵심 개념 3가지 (간략한 설명 포함)
  · 실생활 예시 1가지
  · 형성평가 문제 1개

각 항목을 구분된 슬라이드 구성으로 제안해 줘.
실습 순서
  1. PowerPoint에서 수업 자료 파일을 열고 Claude 애드인 패널을 활성화합니다.
  2. 위 프롬프트 중 하나를 선택해 [ ] 부분을 수정한 뒤 입력합니다.
  3. Claude의 제안을 확인하고 마음에 드는 부분을 슬라이드에 적용합니다.

선생님 실습

9. 선생님 실습 — Excel 플러그인으로 성적·출결 처리

Excel에 Claude를 연결하면 성적 분석, 출결 현황 정리, 통계 생성을 자연어 지시만으로 처리할 수 있습니다.

05
Excel 플러그인 15분

성적·출결 데이터 분석 및 정리

실습 A: 성적 데이터 분석

현재 시트의 학생 성적 데이터를 분석해 줘.

1. 과목별 평균과 표준편차를 계산해 줘.
2. 성적 구간별(90점 이상 / 80~89 / 70~79 / 70점 미만) 학생 수를 정리해 줘.
3. 평균 이하 학생 목록을 별도 시트에 정리해 줘.
4. 결과를 보기 좋은 표 형식으로 정리해 줘.

실습 B: 출결 현황 정리

현재 시트의 출결 데이터를 분석해 줘.

1. 학생별 결석·지각·조퇴 횟수를 집계해 줘.
2. 출석률이 90% 미만인 학생 목록을 별도로 추출해 줘.
3. 월별 출결 현황 요약표를 만들어 줘.
4. 생활기록부 기재에 필요한 형식으로 정리해 줘.
"이 데이터를 학부모 상담 자료로 활용할 수 있게 요약 보고서를 만들어 줘."처럼 자유롭게 추가 요청도 가능합니다.
