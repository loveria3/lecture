🖥️ Claude 데스크탑 앱 설치 가이드

2일차 실습(Skills)을 위한 사전 설치 · 필수 설정 안내

📦 다운로드 ⬇️ claude.ai/download 바로가기
Windows 10 이상 · macOS 지원  ·  Claude Pro 사전 가입 필수
📌
개인 노트북 연결이 어려운 경우

수업 현장 PC(강사 데스크탑 등)에서도 아래 순서를 그대로 따라 하면 동일하게 설치·실습할 수 있습니다. 어떤 PC든 Claude 계정으로 로그인만 하면 설정이 그대로 적용됩니다.

설치

Claude 데스크탑 앱 설치하기

1
claude.ai/download 접속
위의 ⬇️ claude.ai/download 바로가기 버튼을 클릭하거나, 브라우저 주소창에 claude.ai/download를 입력해 접속합니다.
2
운영체제에 맞는 설치 파일 다운로드
현재 PC가 Windows인지 macOS인지 확인 후 해당 설치 파일을 받습니다. (제주고 데스크탑은 대부분 Windows)
3
설치 파일 실행
다운로드된 설치 파일을 더블클릭하고, 안내 창에서 설치를 눌러 진행합니다. 별도 옵션 선택 없이 기본값으로 완료하면 됩니다.
4
Claude 계정으로 로그인
설치 완료 후 앱을 실행하면 로그인 화면이 뜹니다. 사전 가입해 둔 Claude Pro 계정으로 로그인합니다.
🖼️ 설치·로그인 화면 캡처 예정
다운로드 페이지 → 설치 진행 화면 → 로그인 화면 스크린샷을 이 자리에 넣을 예정입니다.
🖼️

설치 단계 스크린샷 추가 예정

필수 설정

설치 후 반드시 켜야 하는 기능

1
설정(⚙️) → 기능 → 코드 실행 및 파일 생성 ON
이 설정이 꺼져 있으면 Skills가 파일(.docx·.xlsx·.pptx·.pdf·.hwpx)을 생성하지 못합니다. 2일차 실습 전 반드시 켜 두어야 합니다.
2
바탕화면 또는 문서함에 새 폴더 만들기
바탕화면(또는 문서함)에서 마우스 오른쪽 클릭 → 새로 만들기 → 폴더. 이름은 "클로드_실습"처럼 알아보기 쉽게 짓습니다. 오늘 실습에서 만드는 모든 파일이 이 폴더에 저장됩니다.
3
Cowork 탭 열기 → 방금 만든 폴더 연결
상단 Chat / Claude Code / Cowork 탭 중 Cowork를 클릭하고, 폴더 연결 버튼을 눌러 방금 만든 폴더를 선택합니다.
4
(선택) 사용자 지정 → 스킬에서 필요한 스킬 확인
한글(HWPX) 스킬처럼 별도 업로드가 필요한 스킬은 좌측 사이드바 사용자 지정 → 스킬에서 추가로 등록합니다.
⚙️ 실제 설정 화면
설정(⚙️) → 기능 → 코드 실행 및 파일 생성
→ 토글 ON
💡
Free · Pro · Team · Enterprise 플랜 모두 Skills 사용이 가능하지만, 코드 실행 및 파일 생성 설정이 켜져 있어야 합니다. 어떤 PC에서 로그인하든 이 설정은 계정 단위로 저장되어 다시 켤 필요는 없습니다 (단, PC가 바뀌면 최초 1회는 확인이 필요합니다).
체크리스트

설치 완료 확인

  • Claude 데스크탑 앱을 설치하고 Pro 계정으로 로그인했다
  • 설정 → 기능 → 코드 실행 및 파일 생성을 ON으로 켰다
  • Cowork 탭에서 작업 폴더를 연결했다
  • Cowork 탭에서 "테스트용 텍스트 파일 만들어줘"라고 요청해 실제 파일이 생성되는지 확인했다
