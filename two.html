3교시 강의 · 15분

Skills — 클로드 데스크탑 앱 확장 기능

Skills는 클로드 데스크탑 앱에 설치하는 추가 기능 패키지입니다. HWPX 문서 생성, Word 자동화 등 웹 버전에서는 불가능한 기능을 사용할 수 있습니다.

Skills란?
클로드 데스크탑 앱에서만 사용 가능한 확장 기능입니다.
설치형 플러그인처럼 필요한 기능만 골라 추가합니다.
파일 생성·수정, 외부 서비스 연동 등 다양한 자동화가 가능합니다.
웹 vs. 데스크탑 앱 차이 웹(claude.ai): 텍스트·이미지 생성 중심
데스크탑 앱: Skills로 파일 직접 생성·수정 가능

1
클로드 데스크탑 앱을 실행합니다.
2
좌측 하단 설정 아이콘(⚙)을 클릭합니다.
3
[Capabilities] 탭에서 Skills를 찾아 설치합니다.

설치 후 새 대화에서 Skills가 자동 활성화됩니다.
교원에게 유용한 Skills

Word Skill — 가정통신문·계획서 등 Word 파일을 자동 생성합니다.

Skills는 Cowork 모드(클로드 데스크탑 앱의 작업 공간 기능)에서 더욱 강력하게 활용할 수 있습니다. 파일 읽기·쓰기·편집을 자연어 지시만으로 처리할 수 있습니다.

3교시 실습 1 · 20분

실습 — HWPX 공문서(결과보고서) 자동 작성

HWPX Skill을 활용해 교육청 제출용 사업 결과보고서를 한글 파일(.hwpx)로 자동 생성합니다.

01
HWPX 공문서 20분

사업 결과보고서 자동 생성

정보만 입력하면 기안문 형식(목적·경위·추진 결과·향후 계획)의 HWPX 파일이 자동으로 생성됩니다.

당신은 [학교명] 진로부 담당 교사입니다.

다음 정보를 바탕으로 교육청 제출용 사업 결과보고서를 HWPX 파일로 작성해 줘.

- 사업명: [사업명]
- 운영 기간: [시작일] ~ [종료일]
- 참여 대상: [학년] [학생 수]명
- 주요 활동 내용: [활동 1], [활동 2], [활동 3]
- 예산 집행 현황: 총 [금액]원 (항목별 명세 포함)
- 성과 및 기대효과: [내용]

기안문 형식으로 수신·제목·본문(목적·경위·추진 결과·향후 계획)·붙임 구조를 갖추고,
공문서 규정에 맞게 A4 2매 이내로 작성해 줘.
실습 순서
  1. 클로드 데스크탑 앱에서 HWPX Skill이 활성화되어 있는지 확인합니다.
  2. 위 프롬프트를 복사해 [ ] 항목을 실제 정보로 수정합니다.
  3. 클로드에 붙여넣기 후 전송하면 .hwpx 파일이 생성됩니다.
  4. 생성된 파일을 한글(HWP)에서 열어 내용을 확인하고 최종 수정합니다.
사업명과 참여 정보만 교체하면 모든 교육 사업의 결과보고서에 반복 사용할 수 있습니다.

3교시 실습 2 · 10분

실습 — 상장·수료증 Word/Excel 병합 자동화

Excel 명단 파일과 Word 상장 템플릿을 활용해 학생별 상장을 일괄 생성합니다.

02
Word 병합 자동화 10분

Excel 명단 → 학생별 상장 일괄 생성

30명 상장을 개별 작성하는 대신, 명단 파일 하나로 전체 상장을 5분 내 자동 완성합니다.

준비물

  • Excel 명단 파일: 이름, 반, 수상 내역, 수상 날짜 열 포함
  • Word 상장 템플릿: [이름], [반], [수상명] 위치에 표시
  • 클로드 데스크탑 앱 + Excel Skill 활성화

생성 결과물

  • 학생별 개별 상장 Word 파일 (또는 페이지별 구분)
  • 수상 내역이 자동으로 채워진 완성본
  • 인쇄만 하면 바로 사용 가능한 상태
Excel 파일의 학생 명단을 읽어서 각 학생별 상장을 자동으로 만들어 줘.

[Excel 파일 첨부 또는 다음 명단 직접 입력]
이름 | 반 | 수상 내역 | 날짜
[학생 1] | [반] | [최우수상] | [날짜]
[학생 2] | [반] | [우수상] | [날짜]

상장 형식:
- 제목: "[수상 내역]"
- 본문: "위 학생은 [학교명] 교육 활동에서 탁월한 성과를 거두었기에 이 상장을 수여합니다."
- 수여자: [교장 선생님 성함]
- 날짜: 2026년 [월] [일]

각 학생별로 구분된 Word 파일로 만들어 줘.
수료증, 감사패 등 다른 문서도 동일한 방식으로 Excel 명단에서 일괄 생성할 수 있습니다.

4교시 강의 · 10분

Claude 애드인 — MS Office 통합 활용

Excel과 PowerPoint에 Claude를 직접 설치하면 현재 파일의 내용을 자동으로 인식해 분석·수정·생성이 가능합니다.

설치 방법
1
Excel 또는 PPT를 열고 [홈] 탭 클릭
2
[추가기능] 버튼 클릭
3
검색창에 "Claude" 입력 후 설치
4
우측 패널에 Claude 창이 열립니다
한 번 설치로 Excel/PPT 모두 사용 Excel에 설치하면 PPT에서도 동일하게 사용할 수 있습니다.
통합 컨텍스트 활용
현재 열린 파일의 내용을 Claude가 자동으로 읽습니다.
파일 내용을 별도 복사 없이 바로 분석·요약할 수 있습니다.
수정 제안을 받아 직접 파일에 적용할 수 있습니다.
활용 예시 Excel 성적표를 열고 "이 데이터에서 평균 이하 학생을 표시해 줘."라고 입력하면 즉시 처리됩니다.
교원 업무 활용 분야
Excel: 성적 분석, 출결 현황 정리, 통계 자동 생성
PPT: 수업 자료 초안 생성, 슬라이드 수정·보완
Word: 공문·보고서 교정 및 내용 보완
절약 효과 반복적인 데이터 입력·정리 작업을 자연어 지시 하나로 대체할 수 있습니다.

4교시 실습 1 · 20분

실습 — PPT 플러그인으로 교육 자료 작성·수정

PPT 파일을 열어 두고 Claude에 지시하면 슬라이드 내용을 자동으로 생성하거나 개선 제안을 받을 수 있습니다.

03
PPT 플러그인 20분

수업 자료 슬라이드 작성·수정·시각화

실습 A: 슬라이드 내용 개선

현재 PPT 파일의 [3번] 슬라이드를 분석해서:

1. 핵심 내용을 3줄로 요약해 줘.
2. 학생이 더 이해하기 쉬운 표현으로 바꿔 줘.
3. 제목을 더 임팩트 있게 수정해 줘.
4. 내용을 시각적으로 표현할 도식 구성을 제안해 줘.

실습 B: 새 슬라이드 생성

다음 내용으로 수업용 PPT 슬라이드를 만들어 줘.

- 주제: [단원명] 핵심 개념 정리
- 학년: [학년]
- 포함 내용:
  · 핵심 개념 3가지 (간략한 설명 포함)
  · 실생활 예시 1가지
  · 형성평가 문제 1개

각 항목을 구분된 슬라이드 구성으로 제안해 줘.
실습 순서
  1. PowerPoint에서 수업 자료 파일을 열고 Claude 애드인 패널을 활성화합니다.
  2. 위 프롬프트 중 하나를 선택해 [ ] 부분을 수정한 뒤 입력합니다.
  3. Claude의 제안을 확인하고 마음에 드는 부분을 슬라이드에 적용합니다.

4교시 실습 2 · 15분

실습 — Excel 플러그인으로 성적·출결 데이터 처리

Excel에 Claude를 연결하면 성적 분석, 출결 현황 정리, 통계 생성을 자연어 지시만으로 처리할 수 있습니다.

04
Excel 플러그인 15분

성적·출결 데이터 분석 및 정리

실습 A: 성적 데이터 분석

현재 시트의 학생 성적 데이터를 분석해 줘.

1. 과목별 평균과 표준편차를 계산해 줘.
2. 성적 구간별(90점 이상 / 80~89 / 70~79 / 70점 미만) 학생 수를 정리해 줘.
3. 평균 이하 학생 목록을 별도 시트에 정리해 줘.
4. 결과를 보기 좋은 표 형식으로 정리해 줘.

실습 B: 출결 현황 정리

현재 시트의 출결 데이터를 분석해 줘.

1. 학생별 결석·지각·조퇴 횟수를 집계해 줘.
2. 출석률이 90% 미만인 학생 목록을 별도로 추출해 줘.
3. 월별 출결 현황 요약표를 만들어 줘.
4. 생활기록부 기재에 필요한 형식으로 정리해 줘.
"이 데이터를 학부모 상담 자료로 활용할 수 있게 요약 보고서를 만들어 줘."처럼 자유롭게 추가 요청도 가능합니다.
